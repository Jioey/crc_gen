# crc_gen
Generates verilog logic for CRC calculation, based on the MPEG-2 algorithm. Configurable CRC and Data widths and polynomial used.

## How to Use
Modify configurations in \__main__ and simply run the file. This code can be modified for other languages, simply modify 3 strings marked A, B, and C.

## Miscellaneous
Helpful tools:  
- Verifiy CRC: [CRCCalc](https://crccalc.com/?crc=&method=CRC-32/MPEG-2&datatype=hex&outtype=hex)  
- Generate Random Hex: [Browserling](https://www.browserling.com/tools/random-hex)
